
\textbf{key}: \texttt{/initials}

All 13 usual free parameters are given initial values. These values act as
the initial parameter positions in some sampling routines, and as the given
value of a parameter, if it is fixed during sampling.

Initial values are stored as attributes to this group.
Any parameters which are not stored here are given default values, listed below.
